# TangkuAgentOS Roadmap

This document describes the planned development direction of TangkuAgentOS.

TangkuAgentOS is being developed as an open-source AI Operating System with modular runtimes, intelligent agents, memory systems, workflows, and extensible AI infrastructure.

---

# 🚀 Version 0.1 — Bootable Core

## Goal

Create a stable foundation where TangkuAgentOS can start, manage runtimes, and provide a reliable development platform.

## Planned Features

- AI Kernel
- Runtime Manager
- Configuration System
- Boot Manager
- Lifecycle Management
- CLI Interface
- Logging System
- Event Bus
- Health Monitoring
- Basic Provider Runtime

Status:

🟡 In Development

---

# 🧠 Version 0.5 — Intelligent Runtime Platform

## Goal

Expand TangkuAgentOS into a complete intelligent runtime environment.

## Planned Features

- Advanced Memory Runtime
- Workflow Engine
- Scheduler
- Plugin System
- Agent Ecosystem Runtime
- Multi-provider AI support
- Tool Registry
- Agent Management
- Improved Testing System

Status:

⏳ Planned
